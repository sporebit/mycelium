## 9. Principles & learnings

- **CSS animations are unreliable for complex backgrounds** — HyphalThreads rewritten SVG/CSS → Canvas. Canvas is the pattern for motion-heavy backgrounds.
- **Workout templates = single source of truth;** day logs are snapshots.
- **Voice captures are context, not data** — for finances especially, CSV is canonical and voice annotates/reconciles.
- **Entity review for automated captures** — voice/Telegram-created entities route to review; UI-created bypass.
- **Open Food Facts:** UK subdomain, English forced; USDA secondary for raw ingredients; Claude Vision OCR fallback for label scanning.
- **Diagnose first, then fix** — after shotgun fixes caused regressions, send a diagnose-first prompt before fixing batches of bugs.
- **True redaction > CSS blur** — never render the secret value in the DOM when hidden; constant-width placeholder; default hidden, session-only reveal.
- **Migration numbering by BUILD ORDER, not feature label** — assign next available number at build time; out-of-order numbers break `supabase db push`.
- **Daily trackers must boundary on Europe/London, not UTC** — BST shifts "today" by an hour; use `toLocaleDateString("en-CA", { timeZone: "Europe/London" })`.
- **One-tap capture needs an undo/delete path** — no-confirm actions create permanent phantom data that erodes trust.
- **Apple Health has no web API** — bridge = Health Auto Export iOS app → REST POST to `/api/health-import`; phone must be unlocked; set to daily aggregation.
- **Vercel cron on Hobby: once/day max** — sub-daily expressions fail the deploy. Workaround: `vercel.json` cron daily as backup; cron-job.org drives sub-daily (free, custom headers, every-minute resolution).
- **App-wide session middleware blocks external machine API callers** — exempt only specific paths; confirm each exempted route enforces its own bearer-token auth.
- **OpenAI Whisper 429 `insufficient_quota` = billing state** — check platform.openai.com → Billing. Not a code bug.
- **Voice capture failure must never silently drop a capture** — reply in Telegram on error + preserve `file_id` for retry.
- **React 19 purity is stricter than 18** — `Date.now()` anywhere in the render path (incl. `useMemo`/`useRef` initialisers) is a build error; fix with `useState(() => Date.now())` (initialisers are exempt). Don't call a state-setting `useCallback` from inside an effect; inline the async fetch as an IIFE with a cancellation flag.
- **Supabase `.or()` ilike filters break on commas/periods in values** — those are the filter delimiters; sanitise alias/input strings before building `.or()` strings. (Latent — matters when the fuller alias management UI lands.)
- **PostgREST silently truncates at 1000 rows** — `.limit(n)` client-side is ignored; the REST layer caps at 1000. Fix: server-side PostgreSQL aggregate function (`txn_agg`) bypasses the cap. Any server-side sum/count over a large table needs this treatment.
- **IS DISTINCT FROM vs !=** — `!=` excludes NULLs; `IS DISTINCT FROM 'x'` treats NULL as "not x". Use `IS DISTINCT FROM` when filtering categories that may be unset.
- **CSS transitions need a forced reflow** — setting React state that applies the end value directly doesn't trigger animation (React batches it). Fix: imperative DOM access after `void el.offsetHeight`. Correct use of DOM manipulation in React (TransitionOverlay).
- **Catmull-Rom via quadratic midpoints** — smooth closed organic curves: move to midpoint between point 0 and point 1, then `quadraticCurveTo` through each successive point to the next midpoint. Continuous curves that look irregular without being spiky.
- **Growing thread animation** — progressive bezier draw (t=0 to growT, step dt=0.02, loop) rather than `setLineDash`; gives full control over tip effects and secondary branching timed to growT.
- **Offscreen canvas for static textures** — grain generated once into an offscreen canvas, blitted each rAF frame. Regenerating 20k+ random dots at 60fps is fatal; blit costs near zero.
